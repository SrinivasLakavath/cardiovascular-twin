Implement a canonical lumped-parameter Windkessel model.

dP/dt = (Q - P/R) / C

Requirements:
- Use numpy and scipy.integrate.odeint
- No plotting
- No randomness
- No ML
- Deterministic output

1) windkessel_ode(P, t, R, C, Q)

- simulate pressure over time

Do NOT add extra features or abstractions.
Return only the code for core.py."
