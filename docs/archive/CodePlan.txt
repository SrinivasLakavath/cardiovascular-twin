## 1. physio_engine/

**Purpose: Physiology-inspired ground truth generator**
⚠️ Touch this ONCE, then freeze.

```
physio_engine/
└── windkessel/
    ├── core.py
    └── __init__.py
```

### core.py

**Responsibility**

* Implements canonical Windkessel model
* Produces SBP & DBP

**Allowed**

* numpy
* scipy

**NOT allowed**

* ML
* drugs
* plotting
* randomness

**Core functions**

```python
def simulate_bp(R, C, Q, t_end=10.0, dt=0.01) -> (sbp, dbp)
```

📌 This file is your **physiology anchor**.
Never mention “toy model” — call it *canonical lumped-parameter model*.

## 2. synthetic_layer/

**Purpose: Turn physiology into a controllable experiment engine**

```
synthetic_layer/
├── patient_sampler.py
├── intervention_mapper.py
└── data_generator.py
```

### patient_sampler.py

**Responsibility**

* Generate synthetic patient profiles

```python
def sample_patient() -> dict
```

Example output:

```python
{
  "age": 55,
  "baseline_sbp": 130,
  "baseline_dbp": 85,
  "hr": 72,
  "risk_group": "medium"
}
```

**Rules**

* Use ranges, not distributions pulled from papers
* Deterministic + seeded randomness

---

### intervention_mapper.py

**Responsibility**

* Abstract drug effects → model parameters

```python
def map_intervention(drug_class, dosage) -> {R, C, Q}
```

**This file is CRITICAL for defense**

You explicitly encode assumptions:

* beta_blocker → ↓Q
* vasodilator → ↓R
* stimulant → ↑Q

This is where **XAI later makes sense**.

---

### data_generator.py

**Responsibility**

* Glue everything together
* Generate dataset rows

Pipeline:

1. sample patient
2. get baseline params
3. apply intervention
4. call simulate_bp
5. compute deltas
6. save row

Output → CSV

📌 This is your **synthetic data factory**.
